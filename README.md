# ESP32 藍芽→紅外線發射器 (bt-ir-sender)

透過 Classic Bluetooth (SPP) 接收手機指令，查詢 IR 對照表後發射對應的 NEC 紅外線訊號。使用 BC337-40 NPN 電晶體驅動紅外線 LED，增強發射距離。

## 硬體需求

- ESP32 DevKit
- IR 紅外線 LED
- BC337-40 NPN 電晶體
- 470Ω 電阻（基極限流）
- 47Ω 電阻（IR LED 限流）

## 電路接線

```
ESP32 GPIO 32 ──[470Ω]── BC337-40 Base (B，中間腳)

5V (VIN) ──[47Ω]── IR LED (+) ── IR LED (-) ── BC337-40 Collector (C，左腳)

BC337-40 Emitter (E，右腳) ── GND
```

> BC337-40 TO-92 封裝，平面朝向你、腳位朝下，從左到右：C (Collector) / B (Base) / E (Emitter)

### 電路原理圖

```
        5V (VIN)
           │
         [47Ω]
           │
       IR LED (+)
           │
       IR LED (-)
           │
           C
    GPIO 32 ──[470Ω]── B   BC337-40
           E
           │
          GND
```

### 電氣參數

| 參數 | 數值 |
|------|------|
| 基極電阻 | 470Ω |
| 基極電流 (Ib) | ≈ 5.5mA |
| 限流電阻 | 47Ω |
| IR LED 電流 (Ic) | ≈ 77mA |
| 電源電壓 | 5V (USB VIN) |
| 總功耗 | ≈ 1.6W |

## 軟體需求

- Arduino IDE（已安裝 ESP32 開發板套件）
- IRremoteESP8266 函式庫
  - Arduino IDE → 工具 → 管理程式庫 → 搜尋 `IRremoteESP8266` → 安裝

## 使用方式

1. 在 Arduino IDE 開啟 `bt-ir-sender.ino`
2. 選擇 ESP32 開發板，編譯上傳
3. 開啟序列埠監控窗（baud: **115200**）
4. 手機安裝 **Serial Bluetooth Terminal** App
5. 手機藍芽設定中配對 `ESP32-BT-Classic`
6. 在 App 中連線並傳送指令

## 指令一覽

### 單一指令

| 指令 | NEC 數據 | 
|------|----------|
| `1` | `0xFFA25D` |
| `0` | `0xFFE21D` |
| `+` | `0xFF22DD` |
| `-` | `0xFFC23D` |
| `m` | `0xFFE01F` |

### 組合指令

| 指令 | 動作流程 |
|------|----------|
| `3` | `1` → 等 0.5s → `0` |
| `4` | `1` → 等 0.5s → `+` → 等 1s → `0` |

## 手機端回饋訊息

| 訊息格式 | 說明 |
|----------|------|
| `OK: 1` | 單一指令執行成功 |
| `combo: 1 -> 0` | 組合指令開始執行 |
| `OK: 3` | 組合指令執行完畢 |
| `ERR: unknown command "x"` | 未知指令 |

## 自訂指令

如需新增或修改 IR 對照表，編輯 `irMap[]` 陣列：

```cpp
const IrMapping irMap[] = {
  { "1", 0xFFA25D },
  { "0", 0xFFE21D },
  // 新增自訂指令
  { "x", 0xYOUR_CODE },
};
```

如需新增組合指令，在 `handleComboCommand()` 函式中加入新的 `if` 區塊。

## 故障排除

| 問題 | 可能原因 |
|------|----------|
| 藍芽搜尋不到裝置 | 確認 ESP32 已上電且序列埠顯示「等待配對連線」 |
| 連線後無法發射 IR | 檢查 BC337-40 的 C/E 腳位是否正確（平面朝你：左C 中B 右E） |
| 發射距離太短 | 確認使用 5V 電源而非 3.3V；檢查 47Ω 電阻值是否正確 |
| 序列埠顯示亂碼 | 確認 baud rate 設為 115200 |
